A child theme called celestine created using Underscores. 

== Description ==

Celestine is a ready to use theme with a clean and minimalistic layout. It has a two column layout, with an author information widget. The latest post and sticky/featured posts are styled differently, so as to emphasize focus on them. which makes it great for blogging, photography, food and travel related blogs. 


== Frequently Asked Questions ==

= Does this theme support any plugins? =

celestine includes support for Infinite Scroll in Jetpack.

= Can the Author information widget area be removed? =

No, it cannot be removed.
